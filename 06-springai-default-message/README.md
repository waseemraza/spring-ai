An LLM takes a prompt consisting of messages. Each message plays a role. 
Following are commonly used roles:

| Role          | Description                                                      |
|---------------|------------------------------------------------------------------|
| User          | The main content of the user query.                              |
| System        | Instruction to LLM how to behave.                                |
| Asistant      | The response from the LLM.                                       |
| Function/Tool | Special instructions to run a function or call an external tool. |
